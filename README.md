## Installation

* Install git from git-scm
* Install node

```
git clone https://github.com/ZombieHippie/cream

cd cream

npm install
```

## Debugging

Use nodemon to restart server on changes

use nodemon with:

```
npm run dev
```

## Testing

```
npm test
```
